# Guides

In-depth tutorials and setup guides for development workflows.

## Available Guides

### [Browser-Use Local](./browser-use-local.md)
Self-hosted browser automation with Ollama for product audits, UI testing, and screenshot verification. No cloud APIs required.

**Key features:**
- Fully local execution with Ollama
- Automated screenshots and GIF recording
- Product audit examples (checkout flows, form validation, responsive design)
- Vision model support for screenshot analysis
